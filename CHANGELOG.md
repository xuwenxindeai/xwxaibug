# 更新日志 (Changelog)

本文档记录 AIBug 项目的所有重要更新。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2026-03-30

### ✨ 新增功能

#### AI 自动修复
- AI 自动分析 Bug 描述
- 自动定位问题代码文件和行号
- 自动生成修复方案
- 一键应用修复
- 自动创建备份文件

#### 用户系统
- JWT Token 认证
- 三种角色权限：
  - 管理员：所有权限
  - 开发：项目配置、Git 操作
  - 测试：提交和查看 Bug
- 会话管理（30 天有效期）
- 默认账户（admin/developer/tester）

#### 项目管理
- 支持多项目管理
- 父项目/子项目结构
- 项目快速切换
- 项目路径显示（~/workspace/格式）

#### Git 集成
- 显示当前 Git 分支
- 获取所有分支列表
- 切换分支功能
- 拉取最新代码
- 显示最近 2 条提交记录

#### Bug 管理
- 提交 Bug（标题、描述、优先级）
- Bug 状态跟踪（待处理/分析中/待修复/已修复/错误）
- 优先级分级（P0 紧急/P1 高/P2 中/P3 低）
- 点击状态快速修改
- Bug 修改历史记录
- 自动分析触发（创建时/轮询）

#### 代码分析
- 智能代码定位
- 问题代码与修复代码对比
- 修复要点说明
- 完整的修复代码展示

#### 定时任务
- 自动轮询未处理 Bug（5 分钟间隔）
- 可配置轮询间隔
- 可启用/禁用自动修复

#### 前端界面
- 响应式设计（支持 PC 和移动端）
- 项目状态卡片
- Bug 统计面板
- 代码高亮显示
- 模态框交互
- 底部版权信息

### 🔐 安全性

- .gitignore 配置（排除敏感文件）
- 配置模板（config.example.js）
- 数据库文件保护
- 日志文件保护
- MIT 开源许可证

### 📖 文档

- 完整的 README.md
- 快速开始指南
- API 接口文档
- 配置说明
- 开发指南
- 贡献指南
- 安全提示

### 🛠️ 技术栈

- **后端**: Node.js + Express
- **数据库**: SQLite (better-sqlite3)
- **认证**: JWT (jsonwebtoken)
- **加密**: bcryptjs
- **定时任务**: node-schedule
- **前端**: Bootstrap 5 + jQuery
- **Git 集成**: 原生 Git 命令

## 版本说明

### 语义化版本

- **主版本号 (Major)**: 不兼容的 API 修改
- **次版本号 (Minor)**: 向下兼容的功能性新增
- **修订号 (Patch)**: 向下兼容的问题修正

### 更新类型

- `新增` - 新增功能
- `改进` - 现有功能的改进
- `修复` - Bug 修复
- `安全` - 安全性修复
- `文档` - 文档更新
- `性能` - 性能优化
